# MeditateOS 1.0
<p><b>MeditateOS 1.0</b> is the first major operating system of meditateOS. First released to the public on Scratch on 14 February 2025, it has been succeeded by MeditateOS 2.0.</p>
## Editions
<p>MeditateOS has 5 editions, 2 of it being remixes of Azhel-Sharp, a cosntructed programming language in Scratch developed by DaRender.</p>
### MeditateOS 1.0 No Media
<p>Developed for the European market (there isn't any European market for Scratch operating systems) as there isn't a built-in media player.</p>
### MeditateOS 1.0 Root
<p>Designed for those who despise bloatware. Just the Trash app and very simple background and 'calmbar' (meditateOS equivalent of the Menu Bar in macOS)</p>
### MeditateOS 1.0 AZ+
<p>Includes the AZ+ (Azhel Sharp) language developed by DaRender. Fix 1 was however added by me.</p>
